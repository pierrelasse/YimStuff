## Quick installation
- Download [yimutils.lua](https://github.com/pierrelasse/YimStuff/releases/download/1.3.9/yimutils.lua) and optionally [SussySpt.lua](https://github.com/pierrelasse/YimStuff/releases/download/1.3.9/SussySpt.lua) too and put them into your scripts folder `%appdata%/YimMenu/scripts`
- Ensure you have yimutils installed.
- Go to `Settings->Lua Scripts` and press `Reload All`
- Enjoy. See [Table of contents](#table-of-contents) for more info.
